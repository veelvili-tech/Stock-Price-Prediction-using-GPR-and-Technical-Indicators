# Stock-Price-Prediction-using-GPR-and-Technical-Indicators

Built hybrid machine learning models to predict defense stock prices using Geopolitical Risk Index (GPR) and technical indicators, providing insights into the impact of geopolitical events on market trends.

Live application deployed using streamlit whicc can be found in the [[link](https://stock-predictor-6-veelvili24.replit.app/)]
